# Spaces — спейсы

> Файл с вопросами для обсуждения. Заполнить ответами и передать в следующую сессию.

## Вопросы

**1. Кто может создать спейс?**
Любой зарегистрированный пользователь? Или есть ограничения (верификация телефона, подписка)?

**2. Как вступить в спейс?**
Способы:
- По инвайт-ссылке
- Поиск публичных спейсов по названию / теме / игре
- Через каталог (как серверы в Discord)
- QR-код

**3. Публичный vs приватный спейс**
Настройка видимости:
- Публичный: виден в поиске, любой может зайти
- По инвайту: виден в поиске, но вход только по ссылке
- Приватный: не виден в поиске, только по инвайту

**4. Структура внутри спейса**
Что можно создавать внутри:
- Текстовые каналы
- Войс-каналы
- Группы
- Категории (папки для группировки каналов, как в Discord)
- Что-то ещё?

**5. Инвайт-ссылки**
Инвайты с ограничением по сроку / количеству использований (как в Discord)?
Кто может генерировать инвайты — только админ или любой участник?

**6. Верификация участников**
Можно ли настроить верификацию при вступлении:
- Подтвердить телефон
- Пройти капчу
- Ответить на вопросы (как скрининг в Telegram-группах)
- Одобрение модератором вручную

**7. Лимит участников**
Есть ли максимум участников в спейсе? Платный план расширяет лимит?

**8. Матчмейкинг внутри спейса**
(Запланировано на потом, но стоит прояснить заранее)
Спейс сможет задавать свои параметры ММ и верификацию рангов (например, через Steam)?
