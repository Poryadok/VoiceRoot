# Roles — ролевая модель и права

> Файл с вопросами для обсуждения. Заполнить ответами и передать в следующую сессию.

## Вопросы

**1. Иерархия ролей — базовая**
Какие роли есть по умолчанию в спейсе:
- Владелец → Администратор → Модератор → Участник → Гость
- Или другая иерархия?

**2. Кастомные роли**
Можно ли создавать свои роли с произвольным набором прав (как в Discord)?
Или фиксированный набор предустановленных ролей?

**3. Права — гранулярность**
Что регулируется правами (примерный список):
- Отправка сообщений
- Загрузка файлов
- Создание каналов
- Управление участниками (кик, бан)
- Выдача ролей
- Запуск ММ от имени спейса
- Включение микрофона в войсе
- Управление записью

**4. Роли в конкретном канале**
Права можно переопределить для отдельного канала (как в Discord: роль X видит этот канал, роль Y — нет)?

**5. Верификационные роли**
Роли, которые выдаются автоматически при верификации (например, "подтверждённый игрок", "Steam-ранк Diamond+") — нужно?

**6. Организатор в войс-чате**
Организатор — это отдельная роль или просто право, которое выдаётся на время сессии?
Организатор может передать управление другому?

**7. DM от незнакомцев**
Кто может писать участникам спейса в DM — все / только участники этого же спейса / только друзья? Настраивается на уровне спейса?
