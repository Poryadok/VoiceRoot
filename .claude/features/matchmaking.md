# Matchmaking — матчмейкинг

## Типы матчмейкинга

| Тип | Статус | Особенность |
|-----|--------|-------------|
| Глобальный | Первая версия | Параметры вводятся вручную, без верификации данных |
| Внутри спейса | Позже | Спейс задаёт параметры и верификацию; меньше аудитория, но лучше фильтрация |

Глобальный — приоритет. Пересмотреть баланс когда аудитория вырастет до уровня Discord.

## UI — расположение

- **Десктоп**: панель справа
- **Мобайл**: отдельная вкладка

## Пати

- **Соло**: один пользователь
- **Группа**: все участники войс-чата в момент запуска ММ автоматически считаются пати
- Запустить поиск может любой участник войса
- Если кто-то вошёл или вышел из войса во время поиска → ММ сбрасывается, предлагается начать поиск заново

## Параметры поиска

Параметры задаются отдельно для каждой игры. Игры — из списка (будет отдельный инструмент для добавления игр).

**Пример (Valorant):**
- Игра: Valorant
- Режим: (определяет количество слотов — например, 5v5 → 5 мест)
- Свой ранг
- Ранги, которые ищешь

Для глобального ММ — поля заполняются вручную, верификация рангов отсутствует (на доверии).

## Флоу поиска

1. Пользователь (один или из войса) запускает поиск с заданными параметрами
2. Во время поиска можно пользоваться приложением в обычном режиме
3. Когда матч найден → **попап у всех участников**: "Матч найден — Принять / Отклонить"
4. Если кто-то отклоняет → поиск продолжается или сбрасывается
5. Все приняли → создаётся временная комната

## Временная комната

- Войс + прикреплённый текстовый чат
- Автоудаляется после того как все вышли
- Запись остаётся в истории ММ

## История матчмейкинга

- Список комнат, в которых участвовал пользователь
- Для каждой комнаты — список участников
- Из истории можно:
  - Добавить пользователя в друзья
  - Забанить пользователя (только для ММ)

## Оценка тиммейтов

- Появляется при **выходе из комнаты**
- Шкала: 1–5 звёзд
- Можно пропустить (для всех или для отдельных участников)
- При оценке 1 или 2 — предложение забанить пользователя
- Средний рейтинг пользователя **виден в профиле**

## Бан

- Забанить можно при оценке (1–2 звезды) или через правую кнопку мыши на пользователе
- Бан только в рамках ММ: забаненный пользователь больше не будет матчиться к тебе
- Не влияет на мессенджерную часть (переписка, войс и т.д.)
