# Screen Share — шара экрана

## Доступность

- Доступна в DM, войс-группе, войс-канале

## Одновременные стримы

- Несколько участников могут шарить одновременно
- Зрители выбирают чей стрим смотреть (как Discord Go Live)

## Что можно шарить (десктоп)

- Весь экран
- Отдельное окно / приложение
- Вкладку браузера
- Системный звук (audio capture) вместе с видео — да

## Качество

- Пользователь выбирает fps и разрешение

## Мобайл

- Смотреть чужую шару на телефоне: да
- Шарить экран с телефона: нет (только с десктопа)

## Remote control

- Нет — только просмотр

## Аннотации

- Отложено (не в первой версии)

## Лимиты

- Одновременных стримов в одном войс-чате: **3**

## Запись шары экрана

- В V1 не реализуется
- В будущем: запись шары вместе с аудио войс-сессии
