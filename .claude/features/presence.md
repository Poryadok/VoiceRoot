## Вопросы

**1. Базовые статусы**
Какой набор системных статусов (как в Discord):
- Онлайн / Не активен / Не беспокоить / Невидимый

**2. Кастомный статус**
Пользователь может написать произвольный текст статуса (как в Telegram / Discord)? С эмодзи?

**3. "В игре" — автоматически?**
Приложение автоматически определяет, что пользователь играет (интеграция с системой / играми) и показывает "Играет в Dota 2"?
Или только ручной ввод?

**4. Кому виден статус?**
Настройка: статус виден всем / только друзьям / никому?

**5. Статус в войс-чате**
Когда пользователь в войсе — это отображается как отдельный статус ("В голосовом чате Спейса X")?

**6. Последний раз онлайн**
Показывать ли "был(а) N минут назад" (как в Telegram)? С возможностью скрыть?
