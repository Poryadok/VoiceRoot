# User Profile — профиль пользователя

> Файл с вопросами для обсуждения. Заполнить ответами и передать в следующую сессию.

## Вопросы

**1. Что видно в публичном профиле?**
Что видит другой пользователь, открыв мой профиль? Например:
- Аватар, имя, @username
- Статус (онлайн/офлайн)
- Общие спейсы
- Средний рейтинг ММ (★)
- Количество матчей / игровая статистика
- Био / описание
- Дата регистрации
- Значок верификации (✅ / ®)

**2. Что есть в личных настройках аккаунта?**
Что пользователь может редактировать в своём профиле?

**3. Видны ли друзья в профиле?**
Список друзей — публичный, только для друзей, или скрытый?

**4. Видна ли история ММ в профиле?**
Публичная статистика матчей (игры, количество, рейтинг) — да/нет/частично?

**5. Баннер профиля?**
Как в Discord — можно поставить фоновое изображение за аватаром?
Бесплатно или только по подписке?

**6. Несколько аватаров / анимированный аватар?**
Анимированный аватар (GIF) — только по подписке или для всех?
