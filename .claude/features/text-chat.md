# Text Chat — текстовый чат

## Сообщения

- **Редактирование**: да, лимит 100 лет (фактически без ограничений)
- **Удаление**: да, у всех, без лимита по времени
- **Форматирование**: Discord-стиль
  - bold `**text**`, italic `*text*`, underline `__text__`, strikethrough `~~text~~`
  - spoiler `||text||`
  - inline code `` `code` ``
  - code block ` ```language ` с подсветкой синтаксиса
  - заголовки `# ## ###`
  - списки `- / 1.`
  - цитаты `>`

## Треды и ответы

| Тип чата | Режим |
|----------|-------|
| DM (личка) | только reply на конкретное сообщение |
| Канал | только треды |
| Группа | треды включаются/выключаются в настройках группы |

## Медиа и вложения

- Файлы и изображения: да (лимиты размера — TBD, отдельное обсуждение хранилища)
- Превью ссылок: да, автоматически
- Голосовые сообщения: да (аудиофайл + встроенный плеер)
- GIF: да
- Стикеры: да — системные паки + пользовательские (загрузка своих паков)
- Emoji: да

## Социальные механики

- Реакции на сообщения: да
- @упоминания: да (завязаны на систему нотификаций — TBD отдельно)
- Закреплённые сообщения: да, как в Telegram

## История и поиск

- История: бесконечная (лимит не установлен на старте, пересмотреть по нагрузке)
- **Поиск из чата** → поиск по тексту и ссылкам внутри этого чата
- **Поиск с главного экрана** → глобальный поиск по всем чатам
- У каждого чата есть раздел shared media: отдельные вкладки для фото / видео / файлов / ссылок

## Статусы доставки

- Как в Telegram: одна галочка = доставлено, две = прочитано
- В группах/каналах — по аналогии с Telegram (счётчик просмотров или индикатор)

## Хранение

- Все сообщения и файлы хранятся на сервере
- Детали файлового хранилища (S3, CDN, лимиты) — отдельное обсуждение
