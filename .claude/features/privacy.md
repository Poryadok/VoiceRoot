# Privacy — настройки приватности

## Общее

Настройки приватности применяются ко всем сущностям приложения. Каждое чувствительное поле имеет собственный контрол видимости/доступа.

При создании профиля пользователь выбирает один из трёх пресетов с предустановленными настройками приватности. Пресет можно поменять позже вручную.

### Таблица дефолтов по пресетам

ДД = Друзья друзей, УС = Участники спейса

| Поле | Личный | Игровой | Рабочий |
|------|--------|---------|---------|
| Онлайн-статус | Друзья | Все | УС |
| Статус «в игре» | Друзья | Все | Никто |
| Рейтинг ММ | Друзья + ДД | Все | Никто |
| Телефон | Никто | Никто | Никто |
| Стори | Друзья + ДД | Все | Никто |
| Поиск по телефону | Друзья | Друзья | Никто |
| Начинать диалог | Друзья + ДД | Все | УС + Друзья |
| Звонить | Друзья | Друзья + ДД | УС + Друзья |
| Приглашать в чаты/спейсы | Друзья | Друзья + ДД | Никто |
| Присылать файлы | Друзья + ДД | Друзья + ДД | УС + Друзья |
| Присылать голосовые | Друзья | Друзья + ДД | УС + Друзья |
| Приглашать в друзья | Все | Все | УС |

**Логика пресетов:**
- **Личный** — полуоткрытый, для реальных друзей
- **Игровой** — открыт к незнакомцам по игровым осям; оптимален для матчмейкинга
- **Рабочий** — закрыт снаружи, открыт внутри спейса; телефон и игровые данные скрыты

## Контрол выбора аудитории

Мультиселект со следующим набором опций (единый для всего приложения):

| Опция | Поведение |
|-------|-----------|
| **Все** | Шорткат вверху списка — снимает «Никто», выбирает все остальные опции |
| **Никто** | Шорткат вверху списка — снимает весь выбор |
| Друзья | Стандартная опция |
| Друзья друзей | Стандартная опция |
| Участники спейса | При выборе появляется второй мультиселект — список спейсов профиля |

«Все» и «Никто» стоят первыми в списке и являются взаимоисключающими шорткатами, а не рядовыми пунктами.

Итоговая аудитория — **объединение** всех выбранных групп.

### Выбор спейсов

Если выбрана опция «Участники спейса», появляется мультиселект со спейсами, в которых состоит пользователь. Доступ получают участники выбранных спейсов.

## Поля с настройками приватности

### Видимость данных

Кто видит информацию:

- Онлайн-статус
- Статус «в игре»
- Рейтинг ММ
- Телефон
- Стори

### Разрешения на действия

Кто может совершать действия:

- Поиск профиля по номеру телефона
- Начинать диалог
- Приглашать в групповые чаты и спейсы
- Звонить
- Присылать файлы
- Присылать голосовые сообщения
- Приглашать в друзья
