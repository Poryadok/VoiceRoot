# Privacy — настройки приватности

## Общее

Настройки приватности применяются ко всем сущностям приложения. Каждое чувствительное поле имеет собственный контрол видимости/доступа.

Дефолтные значения для конкретных шаблонов профилей определяются при обсуждении соответствующих шаблонов.

## Контрол выбора аудитории

Мультиселект со следующим набором опций (единый для всего приложения):

| Опция | Поведение |
|-------|-----------|
| **Все** | Шорткат вверху списка — снимает «Никто», выбирает все остальные опции |
| **Никто** | Шорткат вверху списка — снимает весь выбор |
| Друзья | Стандартная опция |
| Друзья друзей | Стандартная опция |
| Участники спейса | При выборе появляется второй мультиселект — список спейсов профиля |

«Все» и «Никто» стоят первыми в списке и являются взаимоисключающими шорткатами, а не рядовыми пунктами.

Итоговая аудитория — **объединение** всех выбранных групп.

### Выбор спейсов

Если выбрана опция «Участники спейса», появляется мультиселект со спейсами, в которых состоит пользователь. Доступ получают участники выбранных спейсов.

## Поля с настройками приватности

### Видимость данных

Кто видит информацию:

- Онлайн-статус
- Статус «в игре»
- Рейтинг ММ
- Телефон
- Стори

### Разрешения на действия

Кто может совершать действия:

- Поиск профиля по номеру телефона
- Начинать диалог
- Приглашать в групповые чаты и спейсы
- Звонить
- Присылать файлы
- Присылать голосовые сообщения
- Приглашать в друзья
