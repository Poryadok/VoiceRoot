# Friends — друзья и контакты

> Файл с вопросами для обсуждения. Заполнить ответами и передать в следующую сессию.

## Вопросы

**1. Как добавить в друзья?**
Какие способы:
- По @username
- По номеру телефона
- По QR-коду
- Из общего спейса / чата
- Из истории ММ
- Кнопка "добавить" в профиле

**2. Нужно ли подтверждение?**
Добавление в друзья — двустороннее (запрос + принятие) или одностороннее (как подписка)?

**3. Что даёт статус "друг"?**
Какие привилегии по сравнению с просто зарегистрированным пользователем:
- Можно писать в DM без ограничений
- Видно онлайн-статус
- Приоритет в ММ (можно искать вместе)
- Что-то ещё?

**4. Есть ли разные списки?**
Одна категория "друзья" или несколько (например, как в Steam: друзья / заблокированы / избранные)?

**5. Статус заявки**
Видно ли пользователю, что ему отклонили запрос в друзья, или просто "висит" без ответа?

**6. Импорт из телефонной книги**
Люди из телефонной книги, которые уже в Voice — они автоматически попадают в отдельный список или нужно вручную добавлять каждого в друзья?
