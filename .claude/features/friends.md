# Friends & Contacts — друзья и контакты

## Два уровня отношений

| Уровень | Как возникает | Что видит |
|---------|---------------|-----------|
| Контакт | Автоматически из телефонной книги или через поиск | Ник профиля; телефон — только если нашёл по нему |
| Друг | Двустороннее подтверждение (как ВКонтакте) | Больше данных: телефон даже без знания номера, приватные данные по настройкам |

Контакт может отправить запрос в друзья → пользователь принимает → контакт становится другом.

## Добавление в друзья

Способы отправить запрос:
- По @username
- По номеру телефона
- По QR-коду
- Из общего спейса / чата
- Из истории ММ
- Кнопка "добавить" в профиле

Отклонённый запрос **виден** отправителю.

## Контакты из телефонной книги

- Пользователи из телефонной книги, которые есть в Voice, автоматически становятся контактами
- Отдельного списка нет — контакты доступны через поиск
- Можно запретить контактам писать в DM (настройка приватности)

## Незнакомец пишет

Если пользователь, которого нет в контактах и друзьях, пишет в DM:
- Сообщение попадает в **отдельную папку** (как "Запросы сообщений" в Telegram)
- Оповещение помечено как "незнакомец"

## Списки — Steam-модель

- Друзья
- Избранные
- Заблокированные (блокировка только в ММ — отдельная; блокировка в мессенджере — эта)

## Что даёт статус друга

- Быстрый поиск контакта
- Видно онлайн-статус (по настройкам приватности)
- Доступ к бо́льшему объёму данных профиля
- Настройки приватности (сторис, телефон и др.) можно ограничить только для друзей
