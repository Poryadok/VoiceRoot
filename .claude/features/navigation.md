# Navigation — структура и навигация

## Терминология

| Термин в коде | Что это |
|---------------|---------|
| `space` | агрегат-контейнер для каналов и групп |
| `channel` | чат от имени "сущности-канала" (сообщения подписаны каналом) |
| `group` | чат, где пишут участники от своего имени |
| `dm` / `direct` | персональный чат 1-на-1 |

Пользовательский лейбл для `space` — отдельное решение (UI/перевод), в коде и документации используем "пространство" / `space`.

## Модель сущностей-отправителей

- **Пользователь** → имеет аккаунт (identity)
- **Канал** → имеет собственную сущность (name, avatar, описание) — он сам является отправителем
- **Группа** → своей сущности-отправителя нет, пишут конкретные участники
- **Спейс** → не является отправителем напрямую; для объявлений от спейса — выделенный системный канал внутри

## Структура навигации — Десктоп

```
[Колонка активных] [Список чатов папки] [Открытый чат]
  аватар + имя      по выбранной папке
  аватар + имя
  ── папки ──
  Все / ЛС / Группы /
  Каналы / Спейсы /
  + кастомные пользовательские
```

- Левая колонка **всегда видна**, не скрывается при открытии чата
- Активные/закреплённые чаты — вверху колонки
- Лимит на количество активных чатов не установлен (финализировать по отзывам)

## Структура навигации — Мобайл

- Нет открытого чата → полная левая колонка
- Открыт чат → колонка сворачивается в **горизонтальную полоску мини-иконок** вверху экрана
  - Иконки со скроллом (может быть много)
  - На иконках — бейджи непрочитанных
  - Решает проблему "не знаешь, написал ли кто в другой чат пока общаешься"

## Вход в спейс

- **Из папки** → как в десктоп Telegram: название спейса исчезает, остаётся только иконка + открывается колонка с каналами спейса
- **Из колонки активных** → иконка спейса уже там, клик раскрывает список каналов

## Папки по умолчанию

- Все
- ЛС (личные / DM)
- Группы
- Каналы
- Спейсы
- + кастомные (создаёт пользователь)
