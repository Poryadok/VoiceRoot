# Notifications — уведомления

> Файл с вопросами для обсуждения. Заполнить ответами и передать в следующую сессию.

## Вопросы

**1. Какие типы уведомлений есть?**
Примерный список — что из этого нужно:
- Новое сообщение в DM
- @упоминание в группе / канале
- Ответ на моё сообщение (reply)
- Реакция на моё сообщение
- Запрос в друзья
- Кто-то зашёл в войс-канал
- Матч найден в ММ
- Системные (обновления, безопасность)

**2. Каналы доставки**
Какие каналы уведомлений:
- Push на мобайл (когда приложение свёрнуто)
- Push на десктоп (системное уведомление ОС)
- Бейдж / звук внутри приложения
- Email — да/нет?

**3. Гранулярность настроек**
Где и насколько гибко можно настраивать:
- Глобально (один переключатель)
- Отдельно для каждого спейса
- Отдельно для каждого чата/канала
- По типу события

**4. Режим "не беспокоить"**
По расписанию (например, выключить ночью с 23:00 до 8:00)?
Или только ручное включение?

**5. @упоминания — пробиваются сквозь тишину?**
Если чат заглушён — @упоминание всё равно присылает уведомление?
А @everyone / @here — будет такое?

**6. Уведомления в войсе**
Когда пользователь в войс-чате — как ведут себя уведомления? Приглушаются, читаются вслух, показываются оверлеем?
