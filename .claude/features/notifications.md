# Notifications — уведомления

## Типы уведомлений

| Тип | Есть |
|-----|------|
| Новое сообщение в DM | ✓ |
| @упоминание в группе / канале | ✓ |
| Ответ на моё сообщение (reply) | ✓ |
| Реакция на моё сообщение | ✓ |
| Запрос в друзья | ✓ |
| Матч найден в ММ | ✓ |
| Системные (безопасность, обновления) | ✓ |
| Кто-то зашёл в войс-канал | ✗ |

## Каналы доставки

- Push на мобайл (когда приложение свёрнуто)
- Push на десктоп (системное уведомление ОС)
- Звук / бейдж внутри приложения
- Оверлей во время войс-чата (внутри приложения)
- Email — **только для авторизации** (вход, подтверждение), не для событий

## Гранулярность настроек

Можно настраивать:
- Глобально
- Отдельно для каждого спейса
- Отдельно для каждого канала / чата
- По типу события

## Тихие часы (DND по расписанию)

- Можно задать временной диапазон, когда уведомления не приходят (например, 23:00–8:00)

## @упоминания и тишина

- Заглушённый чат: @username-упоминание **пробивает тишину** — включено по умолчанию, можно отключить в настройках
- @everyone: существует, использовать могут только роли с соответствующим правом; тишину пробивает по той же логике

## Уведомления во время войс-чата

- Показываются как оверлей внутри приложения — войс не прерывается
