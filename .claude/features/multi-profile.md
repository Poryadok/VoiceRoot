# Multi-Profile — множественные профили

## Концепция

Один аккаунт может иметь несколько независимых профилей — отдельных "цифровых личностей" (личный, рабочий, игровой и т.д.). Профили не видны друг другу: собеседник знает только тот профиль, через который с ним общаются.

## Количество профилей

- Бесплатно: до 2 профилей
- По подписке: до 5 профилей

## Создание и настройка

- Создаётся из настроек аккаунта: ник, аватар
- Можно привязать дополнительный номер телефона (не основной)
- Один профиль назначается основным — именно он показывается при поиске по номеру
- Значок верификации (✅ / ®) привязан к конкретному профилю, не к аккаунту
- При создании можно выбрать пресет (Личный / Рабочий / Игровой) с предустановленными настройками

## Переключение

- Мгновенное, без перезагрузки приложения
- Десктоп: выпадающее меню / комбобокс
- Мобайл: свайп по аватарке
- Каждый профиль имеет свою цветовую тему для визуальной индикации

## Контакты и друзья

- Контакты и друзья принадлежат профилю, не аккаунту
- Один пользователь может быть другом разных профилей одного аккаунта (собеседник не знает, что это один человек)
- При поиске по телефону показывается основной профиль; после добавления можно перевести контакт в нужный профиль
- **Блокировка** распространяется на весь аккаунт — обойти через другой профиль нельзя

## Спейсы

- Членство в спейсах принадлежит профилю
- Один аккаунт может состоять в одном спейсе разными профилями
- Переключение профиля — глобальное: нельзя сменить профиль "внутри" спейса

## Статусы и уведомления

- У каждого профиля свой статус и свои настройки уведомлений
- Уведомления приходят по настройкам каждого профиля независимо от того, какой активен
- Если на неактивном профиле стоит DND — его уведомления не показываются; если DND нет — приходят

## Матчмейкинг

- ММ запускается от активного профиля
- Смена профиля во время поиска выводит из пати (поиск сбрасывается)
- Игровые данные (ранг, предпочтения) хранятся в профиле, в котором были заданы

## Войс при смене профиля

- Смена профиля не разрывает активный войс-чат
- Новый войс от другого профиля нельзя начать, пока активен текущий звонок

## Монетизация

- Подписка и платежи привязаны к аккаунту
- Красивый ник (@username без #1234) — один в рамках подписки на аккаунт
- Дополнительные красивые ники для других профилей — докупаются отдельно
- Бонусы подписки (например, увеличенный размер файлов) применяются ко всем профилям аккаунта

## Защита от злоупотреблений

- Ограничение на частоту создания/удаления профилей (антиспам)
- При удалении профиля данные архивируются на бэкенде; пользователь этого не видит
