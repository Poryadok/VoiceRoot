# Roles — ролевая модель и права

## Базовая иерархия в спейсе

Владелец → Администратор → Модератор → Участник → Гость

## Кастомные роли

Можно создавать роли с произвольным набором прав (как в Discord).

## Поведение по умолчанию

- **Дефолтная роль**: "Участник" — выдаётся при вступлении в спейс; настраивается владельцем/админом
- **Наследование прав**: модель как в Discord — highest role wins + явный deny перебивает allow; порядок ролей (приоритет) задаётся в настройках спейса

## Права (гранулярно)

- Отправка сообщений
- Отправка ссылок
- Загрузка файлов
- Создание каналов
- Управление участниками (кик, бан)
- Выдача ролей
- Запуск ММ от имени спейса
- Включение микрофона в войсе (для режима рейд/школа)
- Управление записью
- Использование @everyone (все участники канала)
- Использование @here (только онлайн-участники канала)
- Управление каналами (создание, удаление, настройки)
- Управление ролями (создавать/редактировать роли ниже своей по иерархии)
- Изменение настроек спейса

## Права на уровне канала

Права роли можно переопределить для конкретного канала (роль X видит канал, роль Y — нет) — как в Discord.

## Верификационные роли

Роли, выдаваемые автоматически при верификации участника (например, "подтверждён по Steam", "ранг Diamond+").

## Организатор войс-чата

- Это постоянная роль в спейсе
- Любой участник с ролью организатора автоматически имеет права организатора при входе в войс-чат
- Права: управлять микрофонами участников, выдавать/забирать слово, включать режим "поднять руку"

## DM от незнакомцев

Кто может писать пользователю в DM — настраивается **самим пользователем** в настройках приватности (не на уровне спейса).
