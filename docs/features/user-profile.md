# User Profile — профиль пользователя

## Профиль vs Аккаунт

| Сущность | Что это                                                           | Кто видит               |
|----------|-------------------------------------------------------------------|-------------------------|
| Профиль  | Виртуальная личность: аватар, имя, биография, статус              | Другие пользователи     |
| Аккаунт  | Учётная запись: телефон, email, пароль, подписка, устройства, 2FA | Только сам пользователь |

При мультипрофиле у одного аккаунта может быть несколько профилей — см. [multi-profile.md](multi-profile.md).

## Публичный профиль

Что видит другой пользователь, открыв профиль:

- Аватар (анимированный — по подписке)
- Имя и @username
- Значок верификации (✅ / ®) — если есть
- Онлайн-статус (или "был N минут назад")
- Биография / описание (свободный текст)
- Средний рейтинг ММ (★)
- Highlights (закреплённые коллекции стори) — если есть и не скрыты настройками приватности
- Общие спейсы

Список друзей — **скрыт**.
История матчей — **не отображается** в профиле.

## Настройки профиля (редактирует пользователь)

- Аватар (статичный — бесплатно, GIF — по подписке)
- Баннер профиля (фоновое изображение — по подписке)
- Имя отображения
- Биография
- Кастомный статус (по подписке)
- Настройки приватности (видимость данных для разных категорий пользователей)

Настройки приватности — см. [privacy.md](privacy.md).
