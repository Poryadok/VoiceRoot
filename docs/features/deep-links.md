# Deep Links / Sharing — схема URL

Единая схема ссылок для навигации внутри приложения и шаринга наружу.

## Два формата

| Контекст                | Формат                 | Пример                                          |
|-------------------------|------------------------|-------------------------------------------------|
| Веб / внешние платформы | `https://voice.gg/...` | Открывается в браузере, редиректит в приложение |
| Прямой диплинк          | `voice://...`          | Открывается напрямую в приложении               |

Оба формата ведут на одно и то же. Веб-версия определяет, установлено ли приложение, и редиректит; если нет — открывает веб-клиент.

## Схема URL

### Спейс

```
voice://s/{spaceId}
https://voice.gg/s/{spaceId}
```
Открывает главный канал спейса (или экран вступления, если пользователь не участник).

### Канал / группа внутри спейса

```
voice://s/{spaceId}/c/{channelId}
https://voice.gg/s/{spaceId}/c/{channelId}
```

### Конкретное сообщение

```
voice://s/{spaceId}/c/{channelId}/m/{messageId}
https://voice.gg/s/{spaceId}/c/{channelId}/m/{messageId}
```
Открывает чат, скроллит к сообщению и подсвечивает его.

### Профиль пользователя

```
voice://u/{username}
https://voice.gg/u/{username}
```
Открывает публичный профиль. Username — уникальный handle (например, `@vanya`).

### Инвайт в спейс

```
voice://invite/{inviteCode}
https://voice.gg/invite/{inviteCode}
```
Открывает экран принятия инвайта. Если уже участник — редиректит в спейс.

### Личный чат (DM)

```
voice://dm/{userId}
https://voice.gg/dm/{userId}
```
Открывает ЛС с пользователем. Если чата нет — создаёт новый (или предлагает создать, если приватность запрещает DM).

## Поведение при переходе

| Состояние                                       | Поведение                                                       |
|-------------------------------------------------|-----------------------------------------------------------------|
| Приложение установлено и пользователь залогинен | Открывает нужный экран                                          |
| Приложение установлено, не залогинен            | Открывает экран входа, после входа — нужный экран               |
| Приложение не установлено                       | Открывает веб-клиент или Store/Market с предложением установить |
| Нет прав доступа к объекту                      | Показывает ошибку "Нет доступа"                                 |
| Объект удалён                                   | Показывает ошибку "Не найдено"                                  |

## Шаринг

Из любого объекта (сообщение, профиль, спейс) доступна кнопка "Поделиться", которая копирует ссылку `https://voice.gg/...` в буфер обмена.

## UTM-параметры

Поддерживаем UTM для аналитики источников переходов. Все ссылки из кнопки "Поделиться" генерируются без UTM — пользователь добавляет вручную при желании; внутренние интеграции (онбординг, нотификации) проставляют UTM автоматически.

## Диплинки на профиль

Ссылка на профиль строится по `@username`, не по `userId`. Гостевые аккаунты ссылок не имеют — при регистрации пользователь получает username и ссылка становится доступна.
