# Screen Share — шара экрана

## Доступность

- Доступна в DM, войс-группе, войс-канале

## Одновременные стримы

- Несколько участников могут шарить одновременно
- Зрители выбирают чей стрим смотреть (как Discord Go Live)

## Что можно шарить (десктоп)

- Весь экран
- Отдельное окно / приложение
- Вкладку браузера
- Системный звук (audio capture) вместе с видео — да

## Качество и адаптация

- Пользователь выбирает FPS и разрешение при старте стрима
- Адаптация при слабом канале — **автоматически через LiveKit simulcast**: стример отправляет несколько уровней качества (720p / 360p / 180p), каждый зритель получает тот уровень, который тянет его соединение. Отдельной логики не требуется.

## Мобайл

- Смотреть чужую шару на телефоне: да
- Шарить экран с телефона: нет (только с десктопа)

## Пауза стрима

- Стример может поставить шару на паузу без закрытия потока
- При паузе: видео замораживается на последнем кадре, у зрителей отображается иконка "пауза"
- Реализация: `setEnabled(false)` на video track в LiveKit

## Remote control

- Нет — только просмотр

## Аннотации

- Отложено (не в первой версии)

## Лимиты

- Одновременных стримов в одном войс-чате: **3**

## Запись шары экрана

- В V1 не реализуется
- В будущем: запись шары вместе с аудио войс-сессии
