# Voice — Product Vision

## Идея

Discord-like мессенджер с войс-чатом для СНГ-геймеров с уникальной фичей — встроенным матчмейкингом. Не копия Discord, а функциональная замена с тем, чего в Discord нет.

## Целевая аудитория

- **Старт:** геймеры СНГ (Discord блокируется в РФ, люди ищут замену)
- **Потом:** расширение глобально или в корпоратив через self-hosting
- Корпоратив и геймеры — разные ниши, не смешиваем в MVP

## Ключевые отличия от конкурентов

1. **Матчмейкинг** — встроенный подбор напарников по игре и критериям (роль, ранг и т.д.). Многие игры не предоставляют нормального матчмейкинга и тем более войс-чата.
2. **Не блокируется в РФ** — своя инфраструктура
3. **Self-hosting / федерация** — приватные ноды для корпоративных клиентов и сообществ, озабоченных приватностью; в дорожной карте — [PLAN.md](PLAN.md), **Фаза 19**

## Матчмейкинг — механика

**Авто-очередь по критериям:**
1. Пользователь выбирает игру из каталога (Dota 2, CS2, Valorant, и т.д.)
2. Указывает критерии: роль, ранг, регион, количество слотов
3. Встаёт в очередь
4. При наборе нужного состава — система автоматически создаёт матч-отряд (голос + текст)
5. Участники получают уведомление о подборе (**FCM** на Web/Android с [PLAN.md](PLAN.md) Фазы 6; на iOS — после APNs в Фазе 8, до этого — in-app/WebSocket)
6. История матча сохраняется в профиле

**Ограничение без игровых API:** ранг и роль указывает сам пользователь (self-reported). Верификация через Steam API и т.д. — будущее.

## Архитектура (концептуально)

- **Один глобальный сервер** на старте — упрощает разработку и эксплуатацию
- **Приватные комнаты** — внутри глобального сервера, без отдельной инфраструктуры
- **Self-hosting** — отдельный инсталлируемый образ и S2S-федерация (**Фаза 19** в [PLAN.md](PLAN.md)); на старте — один глобальный сервер

## Целевые платформы (приоритет)

1. **Web** — самый быстрый выход, не нужны апп-сторы
2. **Windows** — основная платформа Discord-аудитории
3. **Android / iOS** — после стабилизации продукта

Flutter — единая кодовая база для всех платформ.

**Локализация:** EN + RU в первых фазах (каркас в Фазе 0, экраны MVP в Фазе 1) — см. [features/i18n.md](features/i18n.md).

## Монетизация

Freemium-модель. Детали — в [subscription.md](features/subscription.md):
- Бесплатный план с базовыми функциями
- Подписка $5/мес: расширенные лимиты, больше профилей, кастомный ник, безлимитное хранилище файлов
- Приём платежей: Paddle (международные), CloudPayments (РФ)
- Self-hosting / корпоратив — **Фаза 19** ([PLAN.md](PLAN.md)); модель монетизации для нод определяется отдельно
